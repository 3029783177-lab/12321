# Black White Pixel Platformer

A small static browser game built with HTML Canvas, CSS, JavaScript, and AI-generated black-and-white pixel assets.

## Run

Open `index.html` directly in a browser, or run:

```sh
node server.js
```

Then visit:

```text
http://127.0.0.1:4173
```

## Controls

- Move: `A` / `D` or arrow keys
- Jump: `W`, `Space`, or up arrow
- Restart current level: `R` or the restart button
- Touch controls appear on small screens
